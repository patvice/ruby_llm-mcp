# Rails app for mcp_app example

This app is part of `/Users/pat/dev/repos/ruby_llm-mcp-2/examples/mcp_app`.

From the example root:

```bash
./bin/setup
./bin/reset-data
cd rails_app
bin/rails server
```

Open [http://localhost:3000](http://localhost:3000).

UI styling uses TailwindCSS (loaded via CDN in
`/Users/pat/dev/repos/ruby_llm-mcp-2/examples/mcp_app/rails_app/app/views/layouts/application.html.erb`).
